# ☕ Customer Revenue & Retention Analysis | SQL | Monday Coffee Sales Analysis

## 📌 Project Overview

Designed and executed an end-to-end SQL analytics project for **Monday Coffee**, an online coffee retailer operating since January 2023. The objective was to analyze customer purchasing behavior, revenue trends, and retention patterns to identify the **top three Indian cities** for expanding physical coffee shop operations.

The project involved **database design, data cleaning, transformation, advanced SQL analysis, and business recommendation generation** using MySQL.

---

# 🔧 Key Responsibilities & Technical Implementation

## 1. Database Design & Data Modeling

* Designed normalized relational database schemas for:

  * Customers
  * Products
  * Sales
* Implemented:

  * Primary Keys
  * Foreign Keys
  * Constraints
  * Auto-increment logic
* Developed SQL triggers to automate `TotalAmount` calculations during transaction inserts.
* Maintained referential integrity across transactional datasets.

### Technologies Used

* MySQL
* Relational Database Management
* SQL Constraints & Triggers

# 🧹 Data Cleaning & Quality Validation

  * Null values

# 📈 Key Business Insights

* Identified high-performing cities with strong:

  * Revenue generation
  * Customer acquisition
